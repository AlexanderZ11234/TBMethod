# TBMethod

## Installation & Uninstallation
### Installation
1. **Download** the latest `"TBMethod-<\*version #\*>.paclet"` file to local machine;
2. **Run** `PacletInstall["<*path-to-download*>/TBMethod-<*version #*>.paclet"]`.

### Uninstallation
- **Run** `PacletUninstall["TBMethod"]`
